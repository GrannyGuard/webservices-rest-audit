# Security code review & kwetsbaarheden

Rubric: *"Verbeteronderzoek security"* — **Security code review & kwetsbaarheden** (15 pt).

CodeQL/SonarCloud), met per kwetsbaarheid: vastlegging, onderbouwing,
prioritering en de risico's bij niet-oplossen (o.b.v. valide bronnen).
