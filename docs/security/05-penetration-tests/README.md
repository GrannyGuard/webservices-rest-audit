# Penetration tests (aantonen kwetsbaarheden)

Rubric: *"Verbeteronderzoek security"* — **Penetration tests** (15 pt).

## Inhoud

*(toe te voegen)* Pentestplan gericht op de hoogste risico's uit
[`docs/security/01-security-audit/risico-matrix-full.md`](../01-security-audit/risico-matrix-full.md),
navolgbaar gedocumenteerde uitvoering en bevindingen die het misbruik van de
meest kritische kwetsbaarheden aantonen.
